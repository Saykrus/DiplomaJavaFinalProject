Welcome, to Password Manager!

Setup:

When you first open Password Manager, it will create a config file containing the default password of 'admin'.
So, for the first time, please log in with the password 'admin'.


When your first log in, please be sure to change the password to make the application more secure.
This can be done by through the 'Change Password' menu button after initially logging in.
